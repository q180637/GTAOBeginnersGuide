All images used as the backgrounds on this guide are publicly available from the official Rockstar Games downloads repository on their website here. (Rockstar Games, 2023)

All information about Grand Theft Auto Online represented on this guide has been found either by first hand gameplay or the relevant listings on the Grand Theft Auto Wiki. (GTA Wiki, 2023)

The font family used on this website is Oswald from Google Fonts which has an Open Font License allowing for use in "products & projects - print or digital, commercial or otherwise." (Google Fonts, 2023)

All code used to build the website was written first hand by the developer (Ben Richardson) and only uses default HTML, CSS and JavaScript libraries. Where the developer did not immediately know how to do something (such as the JavaScript for buttons) information was found on W3 Schools (W3Schools, 2023).

The Mobile app was built using Google's Flutter language which is a Dart framework. As the developer was unfamiliar with this programming language support was needed from the official Flutter Documentation (Flutter Dev, 2023)
